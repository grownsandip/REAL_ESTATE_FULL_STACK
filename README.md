# `Running the code`:
## clone the repository
## Open the terminal go to project drectory and go to client folder 
## run npm install for installing all the required dependencies.
## Create three terminal threads running concurrently one for client ,one for api ,and one for socket
## run "npm run dev" in the client terminal  for the client which will open window in local host
## run " node app.js" in api terminal which run the backend server.
## run "node app.js" in socket terminal which runs socket server.
